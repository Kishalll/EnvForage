# Getting Started with EnvForage

Welcome to the future of environment management.

## 1. Installation

Install the CLI tool globally using pip:

```bash
pip install envforage
```

## 2. Audit your system

Run the audit command to detect your OS, GPU, and CUDA versions:

```bash
envforage audit
```

## 3. Generate a setup script

Specify your desired framework, and EnvForage will fetch the exact compatible versions from our matrix:

```bash
envforage generate --framework pytorch --export shell
```

You are now ready to build without dependency conflicts!
